- Riyadus Saliheen, Nawawi 40

### Phase 5: Arabic + Tafsir + Usul рҹ§ 
- Arabic grammar (Nahw, Sarf)
- Tafsir Ibn Kathir
- Usul al-Fiqh, Hadith grading

### Phase 6: Scholar Mindset рҹҺ“
- Madhhab comparison, Ikhtilaf adab
- Comparative religion, Modern issues
